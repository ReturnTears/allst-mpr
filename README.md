## in Eclipse commit code to github
```text
1. install Git plug-in : Help -> Install New Software -> EGit插件地址：http://download.eclipse.org/egit/updates -> 安装后进行重启Eclipse
2. 在Eclipse中配置EGit: Preferences -> Team -> Git -> Configuration : User Settings -> add -> email for your github account
3. Project -> right-hand button -> team -> share project -> git -> commit project to local repository
4. Commit local code to a remote GIT repository
5. create a project in github.com : https://github.com/xxx/xxx.git
6. project -> right-hand button -> team -> remote -> push -> add URI : https://github.com/xxx/xxx.git  | UserName   | PassWord
7. Push Ref Specifications : Source ref: refs/heads/master | destination ref:refs/heads/master  
```
